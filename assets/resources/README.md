# 美术资源目录

## 目录结构

```
resources/
├── textures/          # 纹理图片
│   ├── pets/         # 萌宠图片
│   ├── ui/           # UI素材
│   ├── elements/     # 属性图标
│   └── backgrounds/  # 背景图
├── animations/        # 动画资源
│   ├── pets/         # 萌宠动画
│   └── effects/      # 特效动画
├── audio/            # 音频资源
│   ├── bgm/          # 背景音乐
│   └── sfx/          # 音效
└── prefabs/          # 预制体
    ├── pets/         # 萌宠预制体
    └── ui/           # UI预制体
```

## 使用说明

1. 所有图片建议使用PNG格式，带透明通道
2. 萌宠图片建议尺寸：256x256
3. UI图片建议尺寸：根据实际UI设计
4. 音频建议使用MP3格式，背景音乐需要循环播放
