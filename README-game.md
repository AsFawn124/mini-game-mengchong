# 萌宠大冒险 - 微信小游戏

> 萌系治愈Roguelike微信小游戏

## 🎮 游戏简介

《萌宠大冒险》是一款萌系治愈风格的轻度Roguelike微信小游戏。玩家收集可爱的萌宠，搭配技能，挑战无尽关卡，与好友比拼谁走得更远！

## ✨ 核心特色

- 🐱 **萌系治愈**：精美可爱的萌宠设计，温暖治愈的游戏氛围
- ⚔️ **Roguelike玩法**：每次挑战都是全新体验，技能自由搭配
- 🏆 **社交竞技**：好友排行榜、组队挑战、萌宠交换
- 💰 **多元变现**：激励视频、内购、广告完美结合

## 🚀 快速开始

### 环境要求
- Cocos Creator 3.8+
- Node.js 16+
- 微信开发者工具

### 安装依赖
```bash
npm install
```

### 开发运行
```bash
# 预览模式
npm run preview

# 微信小游戏开发
npm run dev
```

### 构建发布
```bash
npm run build
```

## 🎯 核心玩法

### 1. 收集萌宠
- 50+只萌宠等你收集
- N/R/SR/SSR四种稀有度
- 火/水/草/光/暗五种属性

### 2. 策略战斗
- 3只萌宠组成战队
- 自动战斗+技能选择
- 属性克制+连携技

### 3. 无尽挑战
- 每次挑战地图随机
- 技能Build自由搭配
- 看谁能走得更远

### 4. 社交竞技
- 好友排行榜
- 组队挑战
- 萌宠交换
