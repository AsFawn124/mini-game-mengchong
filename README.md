## 🎮 游戏设计方案：《萌宠大冒险》

### 核心定位
**"萌系治愈 + 轻度Roguelike + 社交竞技"**

### 一句话描述
> 收集萌宠，搭配技能，挑战无尽关卡，和好友比拼谁走得更远！

### 目标用户
- 18-35岁女性（主）
- 喜欢萌系、治愈风格的玩家
- 碎片时间娱乐的上班族/学生

---

## 🎨 游戏玩法设计

### 核心玩法：萌宠自动战斗 + 技能搭配

```
【游戏流程】
选择萌宠(3只) → 进入关卡 → 自动战斗 → 选择技能/装备 → 继续挑战 → 结算奖励
```

### 详细机制

#### 1. 萌宠系统（收集养成）
| 稀有度 | 数量 | 获取方式 | 特点 |
|:---:|:---:|:---|:---|
| N普通 | 20只 | 关卡掉落 | 基础属性 |
| R稀有 | 15只 | 抽卡/活动 | 特色技能 |
| SR史诗 | 10只 | 抽卡/合成 | 强力组合 |
| SSR传说 | 5只 | 限定活动 | 改变战局 |

**萌宠示例**：
- 🐱 火焰喵：火属性，技能"火球术"
- 🐰 冰霜兔：冰属性，技能"冰冻陷阱"
- 🐻 雷霆熊：雷属性，技能"闪电链"
- 🦊 治愈狐：光属性，技能"群体治疗"

#### 2. 战斗系统（自动+策略）
- **自动战斗**：萌宠自动攻击，玩家专注策略
- **技能选择**：每过3波选择1个技能/装备
- **属性克制**：火>草>水>火，光暗互克
- **连携技**：特定萌宠组合触发强力连携

#### 3. Roguelike元素
- **随机关卡**：每次挑战地图、敌人随机
- **技能Build**：数十种技能自由搭配
- **神器系统**：局内随机获得强力神器
- **无尽模式**：看谁能走得更远

#### 4. 社交竞技
- **好友排行榜**：周榜/月榜/总榜
- **组队挑战**：3人组队，共享技能
- **萌宠交换**：好友间交换碎片
- **竞技场**：异步PVP，防守阵容

## 💰 变现设计（快速赚钱）

### 变现模式组合

| 模式 | 占比 | 说明 |
|:---:|:---:|:---|
| **激励视频** | 40% | 复活、双倍奖励、抽卡 |
| **内购** | 35% | 萌宠抽卡、皮肤、月卡 |
| **Banner广告** | 15% | 底部横幅，非强制 |
| **插屏广告** | 10% | 关卡间自然插入 |

### 具体设计

#### 1. 抽卡系统（核心付费点）
```
单抽：100钻石（约1元）
十连抽：900钻石（9折）
保底机制：30抽必出SR，100抽必出SSR
```

#### 2. 月卡/季卡（稳定收入）
- **月卡**：30元，每日领100钻石+1抽
- **终身卡**：198元，永久每日领150钻石

#### 3. 皮肤系统（外观付费）
- 萌宠皮肤：改变外观+轻微特效
- 场景皮肤：战斗背景个性化
- 价格：6-68元不等

#### 4. 战令系统（留存+付费）
- 免费版：基础奖励
- 进阶版：68元，超值奖励
- 精英版：128元，限定SSR萌宠

```
【主界面布局】
┌─────────────────────────────┐
│  [头像]  昵称    💎999  ⚡50  │  ← 顶部信息栏
```

### 关键界面设计

#### 1. 战斗界面
- 上半屏：萌宠自动战斗动画
- 下半屏：技能选择区（Roguelike三选一）
- 右侧：波数进度、暂停按钮

#### 2. 抽卡界面
- 中央：萌宠展示台（3D旋转）
- 下方：单抽/十抽按钮
- 特效：彩虹光效、粒子爆炸

#### 3. 背包界面
- 网格布局：萌宠卡片展示
- 筛选：按属性/稀有度筛选
- 详情：点击弹出萌宠详情弹窗
