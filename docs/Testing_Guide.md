## 性能测试

### 1. 内存测试
```
测试步骤:
1. 进入游戏，记录初始内存
2. 进行10场战斗
3. 抽卡50次
4. 频繁切换场景
5. 记录最终内存
6. 检查是否有内存泄漏
```

### 2. 帧率测试
```
测试步骤:
1. 开启FPS显示
2. 主场景帧率应保持在58-60fps
3. 战斗场景帧率应保持在55-60fps
4. 特效较多时不应低于45fps
```

### 3. 加载时间测试
```
测试步骤:
1. 冷启动游戏，记录加载时间
2. 场景切换时间应小于1秒
3. 抽卡动画流畅
```
