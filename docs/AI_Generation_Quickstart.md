# AI生成资源使用指南

## 🚀 快速开始 (3步生成)

### 第1步: 生成UI设计 (5分钟)

**工具**: Galileo AI (https://www.usegalileo.ai/)

**操作**:
1. 注册账号 (免费10次)
2. 复制下方Prompt
3. 点击生成
4. 导出Figma文件

**Prompt**:
```
Mobile game UI for "Pet Merge Adventure", 
canvas 750x1334px, cute kawaii style,
pink #FF9A9E and mint #A8E6CF color scheme,
top bar with gold coins, diamonds, energy icons,
center 5x5 white grid for pet merging,
bottom 5 navigation buttons (home, bag, gacha, rank, settings),
rounded corners, soft shadows, flat design,
pastel cream background, decorative stars and hearts
```

**产出**: 完整UI设计稿 (可直接给程序员)

---

### 第2步: 生成萌宠原画 (10分钟)

**工具**: Midjourney (https://www.midjourney.com/)

**操作**:
1. 进入Discord
2. 输入 `/imagine`
3. 粘贴下方Prompt
4. 等待生成 (1分钟)
5. 选择最佳结果

**Prompt** (小仓鼠示例):
```
A cute chibi hamster, round fluffy body, small pink ears, 
big sparkling eyes, holding sunflower seed, orange and white fur,
white background, game character, 2D illustration, 
kawaii style, soft pastel colors --ar 1:1 --niji 5
```

**批量生成** (10只萌宠):
```
# 复制到Midjourney，一次生成一只

1. cute chibi hamster, orange and white, holding seed --ar 1:1 --niji 5
2. cute chibi rabbit, white with pink ears, holding carrot --ar 1:1 --niji 5  
3. cute chibi cat, orange tabby, playful pose --ar 1:1 --niji 5
4. cute chibi dog, Shiba Inu, tongue out --ar 1:1 --niji 5
5. cute chibi fox, red fur, fluffy tail --ar 1:1 --niji 5
6. cute chibi bear, brown, round body --ar 1:1 --niji 5
7. cute chibi panda, black and white --ar 1:1 --niji 5
8. cute chibi penguin, black and white --ar 1:1 --niji 5
9. cute chibi dragon, red scales, small wings --ar 1:1 --niji 5
10. cute chibi unicorn, white with rainbow mane --ar 1:1 --niji 5
```

**产出**: 10张萌宠原画 (PNG格式)

### 第3步: 生成图标 (5分钟)

**工具**: Leonardo.ai (https://leonardo.ai/) 免费版

**操作**:
1. 注册账号 (每日150积分)
2. 选择 "Image Generation"
3. 粘贴Prompt
4. 生成并下载

**Prompt** (首页图标):
```
Game UI icon, cute house shape with heart, 
rounded corners, flat design, pink color,
white background, simple, 64x64px
```

**批量生成** (5个导航图标):
```
1. Game UI icon, cute house, pink, home symbol --ar 1:1
2. Game UI icon, cute backpack, mint green, bag symbol --ar 1:1
3. Game UI icon, crystal ball, purple, magic symbol --ar 1:1
4. Game UI icon, trophy, gold, rank symbol --ar 1:1
5. Game UI icon, gear, gray, settings symbol --ar 1:1
```

**产出**: 5个导航图标 (PNG格式)

## 💡 提高效率技巧

### 1. 批量生成
- Midjourney一次生成4张变体
- Leonardo.ai一次生成4张
- 选择最佳，丢弃其他

### 2. 风格统一
- 所有Prompt加相同后缀
- 使用相同艺术家风格
- 固定色彩描述

### 3. 快速筛选
- 第一眼不好就丢弃
- 不要过度纠结细节
- AI生成很快，重开更省时间

### 4. 后期微调
- 用Photoshop调整颜色
- 用Remove.bg抠图
- 用Upscayl放大图片

## ⚠️ 常见问题

### Q: 生成的图片有版权吗？
A: 
- Midjourney: 付费会员有商用权
- Leonardo.ai: 有商用权
- Galileo AI: 有商用权
- **建议**: 购买付费版确保权益

### Q: 风格不统一怎么办？
A:
- 使用相同Prompt结构
- 固定后缀参数
- 选择相似风格的参考图
- 后期统一调色

### Q: 生成质量不稳定？
A:
- 多生成几次选最佳
- 调整Prompt描述
- 使用参考图垫图
- 降低期望，接受AI局限性

### Q: 需要修改细节？
A:
- 用Photoshop手动调整
- 或用Inpainting局部重绘
- 或重新生成整体
