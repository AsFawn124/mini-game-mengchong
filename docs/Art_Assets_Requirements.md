### 美术规范

#### 萌宠原画
- **画布尺寸**: 512px x 512px
- **导出尺寸**: 256px x 256px (游戏内)
- **格式**: PNG (透明背景)
- **风格**: Q版萌系，2.5头身
- **色彩**: 明亮饱和，渐变阴影

#### 进化形态 (每只3个等级)
| 等级 | 名称后缀 | 变化 |
|:---:|:---|:---|
| Lv.1 | 幼年期 | 小巧可爱 |
| Lv.2 | 成长期 | 体型变大，增加装饰 |
| Lv.3 | 完全体 | 华丽特效，光环加持 |

#### 示例：小仓鼠进化
```
Lv.1: 小小仓鼠
- 体型: 掌心大小
- 特征: 圆滚滚，大眼睛
- 动作: 抱着瓜子

Lv.2: 仓鼠勇士  
- 体型: 足球大小
- 特征: 戴上小头盔，披风
- 动作: 站立姿势

Lv.3: 仓鼠王
- 体型: 篮球大小
- 特征: 皇冠、披风、权杖
- 特效: 金色光环
```

## 📦 资源命名规范

### 萌宠
```
pet_{id}_{level}.png
例如: pet_001_1.png, pet_001_2.png, pet_001_3.png
```

### 背景
```
bg_{scene}.jpg
例如: bg_main.jpg, bg_battle_forest.jpg
```

### UI元素
```
ui_{type}_{name}.png
例如: ui_btn_primary.png, ui_card_pet.png
```

### 图标
```
icon_{name}.png
例如: icon_home.png, icon_gold.png
```

### 特效
```
effect_{name}.png / effect_{name}.plist
例如: effect_merge.png, effect_star.plist
```

### 音效
```
sfx_{name}.mp3 / bgm_{name}.mp3
例如: sfx_button_click.mp3, bgm_main.mp3
```
