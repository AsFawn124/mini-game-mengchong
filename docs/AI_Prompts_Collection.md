## 🎨 七、风格统一技巧

### 保持风格一致的方法:

1. **固定后缀**
   - 所有Prompt结尾加: `--ar 1:1 --niji 5 --style cute`

2. **固定描述词**
   - 始终包含: "chibi style, kawaii, game asset, 2D illustration"

3. **固定色彩**
   - 使用相同色板: pastel colors, soft shading

4. **固定背景**
   - 统一用: "white background"

5. **固定比例**
   - 萌宠: `--ar 1:1`
   - 背景: `--ar 9:16`
   - 图标: `--ar 1:1`
