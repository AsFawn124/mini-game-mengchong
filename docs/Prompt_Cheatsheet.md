### 智谱API调用（CogView-3）

```python
# 安装SDK
pip install zhipuai

# 生成代码
from zhipuai import ZhipuAI

client = ZhipuAI(api_key="你的API_KEY")

pets = ["hamster", "rabbit", "cat", "dog", "fox"]

for pet in pets:
    response = client.images.generations(
        model="cogview-3",
        prompt=f"Cute chibi {pet}, game character, round body, big eyes, kawaii style, white background, 2D illustration",
        size="256x256",
        n=4
    )
    
    # 保存图片
    for i, img in enumerate(response.data):
        url = img.url
        # 下载保存
        print(f"Generated {pet}_{i}: {url}")
```

## 💡 生成技巧

### 1. 豆包使用技巧
- 描述越详细，结果越好
- 使用具体数字（750x1334）
- 提供颜色代码（#FF9A9E）
- 分点描述更清晰
- 一次生成一张，质量更高

### 2. 智谱API使用技巧
- 使用英文Prompt效果更好
- 批量生成节省时间
- 设置n=4一次生成4张选最佳
- 添加size参数控制尺寸

### 3. 通用技巧
- 生成多张选择最佳
- 用PS微调细节
- 保持风格统一
- 命名规范保存
